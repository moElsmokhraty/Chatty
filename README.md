Chatty
=============================
Chatty is a simple chat app that uses Firebase as a backend and provides a simple UI for chatting with friends.

- Flutter
- Cubit
- Firebase Auth
- Firestore Database
- Shared Preferences
- Firebase Cloud Messaging (FCM)
